Для начала нужно установить на свой компьютер последнюю версию языка.

  • в папке header лежит заголовочный файл __"Ве_крест_крест.h"__, который необходимо скачать;
  • далее нужно создать проект на языке C++, под видом которого долгие годы прятался В††. В папку этого проекта, в которой находятся файлы сборки, переносим файл "Ве_крест_крест.h";
